## 功能（Features）
这个库包含的主要内容有：

1. [起始页splash]

2. [第一次导航guidepage]

3. [广告banner]

4. [tab导航和分栏]

5. [定位数据库]

6. [搜索框和界面]

7. [自定义常用控件]

8. 一些常用的工具类，utils

9. [标签布局TagView]

10. 联动布局LInkageView

其中字体、颜色和背景的样式均可自定义，一键设置。
